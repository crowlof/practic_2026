# Численное воспроизведение модели водной экосистемы NPZ

Данный проект реализует модель «биоген–фитопланктон–зоопланктон» (NPZ) с замыканием, описанную в статьях:

- Edwards A. M., Brindley J. Zooplankton mortality and the dynamical behaviour of plankton population models // Bulletin of Mathematical Biology. 1999. Vol. 61. P. 303–339.
- Edwards A. M., Bees M. A. Generic dynamics of a simple plankton population model with a non-integer exponent of closure // Chaos, Solitons & Fractals. 2001. Vol. 12. P. 289–300.

## Зависимости

- Python 3.10+
- Библиотеки: numpy, scipy, matplotlib (версии указаны в `requirements.txt`)

## Установка и запуск

Запустите Jupyter Notebook:
jupyter notebook notebooks/analysis.ipynb
